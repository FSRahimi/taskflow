# TaskFlow - Smart Todo App

TaskFlow is a simple todo app built with React.  
It helps you add tasks, manage them, and track your progress in an easy way.

## Features

- Add tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks (All / Completed / Pending)
- View task details
- Search tasks
- Dark / Light mode
- Saves data in local storage
- Responsive design

## Pages

- Home page (shows intro and stats)
- Tasks page (main workspace)
- Task details page
- Completed tasks page
- 404 page

## Tech used

- React
- React Router
- JavaScript
- Tailwind CSS
- Local Storage

## How to run

npm install
npm run dev
